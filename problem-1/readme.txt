DESCRIPTION
	This tiny program is used
	to get the sum of all the
	multiples of 3 and 5.

SYNOPSYS
	./problem-1 [OPTIONS]... [RANGE]...

RANGE
	When no range specified program
	calculates the sum of all the
	multiples of 3 and 5 from 0 to 1000.

	[number]
		Calculate sum below specified
		number

	[number1]-[number2]
		Calculate sum starting from
		number1 to number2

OPTIONS
	-?, --help
		Output this help message.
